# 🚀 Despliegue en Streamlit Cloud

## Pasos para Deployar

1. Ve a https://streamlit.io/cloud
2. Click en "New app"
3. Selecciona:
   - Repository: senarzuniga/CustomerQualification
   - Branch: main
   - Main file path: streamlit_app.py
4. Click "Deploy"

## Variables de Entorno

En Streamlit Cloud, ve a:
- Settings (engranaje)
- Secrets
- Agrega tu OPENAI_API_KEY
